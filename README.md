drsync
======

Developer: Sammy Fung <sammy@sammy.hk>

Usage
-----

$ git clone https://github.com/sammyfung/drsync    

$ cd drsync/scripts; chmod a+ *.sh   

$ sudo cp drsync.conf-sample /etc/drsync.conf   

$ sudo vim /etc/drsync.conf
